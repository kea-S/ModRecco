# ModRecco

ModRecco is a module recommendation system designed to assist National University of Singapore (NUS) students in discovering relevant modules. It leverages the NUSMods API to provide personalized suggestions based on various criteria.

## Features

*   **Personalized Module Recommendations:** Get module suggestions tailored to your academic profile or interests.
*   **NUSMods Integration:** Utilizes the comprehensive NUSMods API for up-to-date module information.
*   **User-Friendly Interface:** (Once implemented) An intuitive way to explore and filter recommendations.
